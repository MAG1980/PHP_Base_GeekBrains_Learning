ctrl+shift+F7 - поиск вхождений выделенного текста
F3 и shift+F3 - навигация по вхождениям

ctrl+D - дублировать выделенную строку

ctrl+R - поиск с заменой

Alt+Enter - настройки форматирования выделенного текста

[CTRL] + [ALT] + L и IDE отформатирует код в соответствии с настройками (Preferences -> Project Settings -> Code Style -> PHP)

ctrl + F12 - terminal

Shift+F6 - рефакторинг
Вы можете легко переименовывать свои классы, методы и переменные с автоматическим исправлением всех мест, где они используются. Поместите курсор на символ, который хотите переименовать, и нажмите Shift+F6 (Рефакторинг | Переименовать). Введите новое имя и нажмите Enter.
